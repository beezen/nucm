# 更新版本

轻松将工具版本更新到最新。

## 命令

```bash
$ nucm update [options]
```

## 参数

- `options`：可选参数

| 选项       | 描述             |
| ---------- | ---------------- |
| `--silent` | 表示静默更新方式 |

## 示例

1、询问式更新。检查最新的工具版本，并询问是否需要更新。

```bash
$ nucm update

更新检查中......
? 存在可更新的版本。
🌟 nucm  1.8.0  →  1.8.7 (Y/n)
```

2、静默更新。检测到可更新版本时，会自动更新到最新，适用于自动化 CI/CD 流程。

```bash
$ nucm update --silent
```
